# sai-portfolio

Personal portfolio (static site) for Sairam Odela.

Contents
- `index.html` — main page
- `style.css` — styles
- `script.js` — interactive behaviors

Run locally
Serve the folder with a static server, for example:

```powershell
# From the project root
python -m http.server 8000

# then open http://localhost:8000 in your browser
```

Notes
- External assets are loaded from CDNs (Google Fonts, Font Awesome, AOS).
- Links that open in a new tab use `rel="noopener noreferrer"` for safety.
